# Customer Churn Prediction Using Machine Learning

This project evaluates multiple machine learning and deep learning models for predicting customer churn using the IBM Telco Customer Churn dataset.

## Research Paper
The full research paper for this project can be found here:

[Customer Churn Prediction Paper](churn-prediction-ml-comparison.pdf)

## Project Overview
Customer churn prediction helps businesses identify customers who are likely to stop using their service. This study compares traditional machine learning models with a neural network under identical preprocessing and evaluation conditions.

## Dataset
IBM Telco Customer Churn Dataset  
7032 customers  
Churn rate: 26.6%

## Models Evaluated
- Logistic Regression
- Random Forest
- Gradient Boosting
- Neural Network (MLP)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Results
Gradient Boosting achieved the highest ROC-AUC score (0.839), demonstrating strong predictive performance for structured telecom churn data.
